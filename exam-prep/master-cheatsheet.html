Exam Traps

BigQuery Common Mistakes

  • Partition pruning fails when using functions on partition column: WHERE DATE(timestamp_col) = '2023-01-01' - Use WHERE timestamp_col >= '2023-01-01' AND timestamp_col < '2023-01-02'
  • External tables cannot: use DML, streaming inserts, or native partitioning
  • Clustering column order matters: Most queried column first, not highest cardinality
  • BI Engine ≠ Slot reservations: BI Engine is for dashboard acceleration, not general compute
  • Materialized views auto-refresh within 5 minutes for base table changes

Dataflow Gotchas

  • Windowing before GroupByKey: Must window unbounded data before aggregation
  • Side inputs: Use for small reference data, broadcast to all workers
  • Streaming vs Batch: Different APIs and behaviors for same transforms
  • Dataflow Shuffle: Managed service reduces hotspots, use --enable_prime_shuffle
  • Autoscaling delay: 2-5 minutes to scale up, plan for traffic spikes

Security & IAM Traps

  • Dataset-level vs Table-level permissions: Dataset access can grant table access
  • Service account keys: Avoid downloading, use workload identity instead
