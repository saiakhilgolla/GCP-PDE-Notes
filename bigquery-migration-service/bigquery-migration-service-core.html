BigQuery Migration Service

Core Concepts

Overview

BigQuery Migration Service (BQMS) is a serverless, fully-managed service that helps migrate data warehouses from on-premises or other cloud platforms to BigQuery.

Key Components

  • SQL Translation API: Converts SQL from source dialects to BigQuery SQL
  • Migration Assessment: Analyzes source system complexity and readiness
  • Batch Migration: Large-scale SQL translation for entire codebases
  • Interactive Translation: Real-time translation for development/testing
  • Schema Mapping: Converts source schemas to BigQuery format

Architecture

Migration Workflow Architecture

Phase Component Purpose Output
1. Assessment Migration Assessment API Analyze source complexity Readiness report, effort estimates
2. Schema Migration Schema Translation Service Convert DDL statements BigQuery dataset/table schemas
3. SQL Translation SQL Translation API Convert DML/queries BigQuery-compatible SQL
4. Data Movement External tools (DTS, Dataflow) Transfer actual data Populated BigQuery tables
5. Validation Data Validation Service Compare source vs target Validation reports

Integration Points

  • Cloud Storage: Stores source files and translation outputs
  • BigQuery: Target destination for schemas and data
  • Data Transfer Service: Handles data movement from sources
  • Dataflow: Custom data transformation pipelines
  • Cloud SQL: Staging area for complex transformations

Exam Traps

Common Misconceptions

  • TRAP: Data Movement: BQMS does NOT move data - only translates SQL and schemas
  • TRAP: Real-time Migration: No real-time/streaming migration - batch processing only
  • TRAP: 100% Automation: Manual review and testing required for complex migrations
  • TRAP: Bidirectional: Translation is one-way only (source → BigQuery)

Service Confusion

Service Purpose What It Does What It Doesn't Do
BigQuery Migration Service SQL/Schema translation Convert SQL syntax and DDL Move actual data
Data Transfer Service Data movement Copy data from sources Translate SQL code
Database Migration Service Database replication Migrate entire databases Translate to BigQuery
Dataflow ETL processing Transform data in transit Static SQL translation
