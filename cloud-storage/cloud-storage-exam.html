Core Concepts

Object Storage Service

Unified object storage for live data and archival. Objects stored in buckets with global namespace. RESTful API access.

Key Components

  • Bucket: Top-level container with global unique name
  • Object: Individual file with metadata
  • Storage Class: Performance/cost tier
  • Lifecycle: Automated object management

Exam Traps

🚨 Common Gotchas

Storage Class Transitions

  • TRAP: Can only transition to "colder" classes, never warmer
  • TRAP: Early deletion charges apply if deleting before minimum duration
  • TRAP: Retrieval charges apply for Nearline/Coldline/Archive even for lifecycle transitions

Retention & Compliance

  • TRAP: Bucket Lock is permanent and cannot be removed
  • TRAP: Retention policy prevents deletion/modification, not access
  • TRAP: Legal holds can be applied/removed but retention policy cannot

Versioning

  • TRAP: Versioning charges for ALL versions, can get expensive
  • TRAP: Lifecycle rules can delete old versions, but need explicit configuration
  • TRAP: Disabling versioning doesn't delete existing versions

CMEK/CSEK

  • TRAP: CMEK keys in Cloud KMS must be in same region as bucket for regional buckets
  • TRAP: CSEK keys must be provided for every operation on the object
  • TRAP: Losing CSEK key = data is unrecoverable

Transfer Options

  • TRAP: Transfer Appliance for 10TB+, not for small datasets
  • TRAP: Storage Transfer Service for cloud-to-cloud, not on-premise
  • TRAP: gsutil for on-premise to GCS

Access Control

  • TRAP: Uniform bucket-level access disables ACLs permanently
  • TRAP: Signed URLs work without Cloud IAM authentication
  • TRAP: Requester Pays - requester needs BOTH bucket access AND billing permissions
Quick Cheatsheet

Storage Classes

  • Standard: Hot data, no minimum
  • Nearline: 30-day minimum
  • Coldline: 90-day minimum
  • Archive: 365-day minimum

Lifecycle Actions

  • SetStorageClass
  • Delete
  • AbortIncompleteMultipartUpload

Lifecycle Conditions

  • age - days since creation
  • createdBefore - date
  • matchesStorageClass
  • numNewerVersions

Security Options

  • CMEK: Cloud KMS managed
  • CSEK: Customer-supplied
  • Signed URLs: Time-limited access
  • Bucket Lock: Immutable retention

Transfer Tools

  • gsutil: CLI, on-premise to GCS
  • Storage Transfer Service: Cloud-to-cloud
  • Transfer Appliance: 10TB+ offline
  • BigQuery Data Transfer: Analytics data

Key Commands

  • gsutil cp - copy objects
  • gsutil rsync - sync directories
  • gsutil lifecycle set
  • gsutil versioning set on

💡 Memory Aids

  • "NEAR-ly monthly" = Nearline 30 days
  • "COLD quarterly" = Coldline 90 days
  • "ARCHIVE yearly" = Archive 365 days
